## 프로젝트 소개
렌즈 전문 매장 검색 및 위치 기반 서비스 플랫폼의 QA 개선 및 추가 기능 개발 프로젝트입니다.

## 주요 개발 내용
### UI/UX 개선
- 메인 페이지 슬라이더 반응형 디자인 구현
- 네비게이션 반응형 레이아웃 최적화

### 지도 기능 구현
- 카카오 맵 API 통합
- 위치 기반 상점 검색 시스템 개발
- 사용자 위치 기반 자동 상점 표시 기능
- 상점 필터링 및 검색 기능 구현

## 기술 스택
- Frontend: Next.js, React
- 상태관리: Redux, Redux-Saga
- Maps API: Kakao Maps API

## 주요 성과
1. 반응형 디자인 개선으로 모바일 사용성 향상
2. 위치 기반 서비스 도입으로 사용자 편의성 증대
3. 상점 검색 최적화로 매장 접근성 개선
